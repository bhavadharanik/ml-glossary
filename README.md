# ML Engineer Glossary

Interactive glossary covering core ML/AI concepts for interview preparation.

## Features

- **70+ terms** across 8 categories: ML Fundamentals, Deep Learning, NLP & LLMs, RAG, MLOps, Math, System Design, Python
- **Flashcard mode** — test yourself with spaced repetition
- **Quiz mode** — multiple choice questions on each concept
- **Mastery tracking** — tracks which terms you've reviewed (stored in browser)
- **Search** — instant filter across all terms
- **PDF version** — printable reference sheet

## Usage

Open `index.html` in any browser. No server, no dependencies, no setup — just a single HTML file.

Or use the [PDF version](ML_ENGINEER_GLOSSARY.pdf) for offline/print reference.

## Live Site

Enable GitHub Pages (Settings → Pages → Deploy from main branch) to get a live URL.

## Who is this for

Anyone preparing for ML Engineer / AI Engineer interviews. Each term includes:
- **Definition** — what it is
- **Why it matters** — practical relevance
- **Interview tip** — how to talk about it
- **Engineering analogy** — maps ML concepts to software engineering terms
