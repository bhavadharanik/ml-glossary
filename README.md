# ML & AI Engineer Glossary — Interview Prep

Interactive glossary covering core ML/AI concepts for cracking AI Engineer and ML Engineer interviews, with a focus on the UK market.

## Features

- **90+ terms** across 11 categories: ML Fundamentals, Evaluation Metrics, Classical Models, Deep Learning, NLP, Transformers, LLMs & GenAI, RAG & Agentic AI, MLOps, AI Ethics & UK Market, Math
- **Flashcard mode** — test yourself with active recall
- **Quiz mode** — multiple choice self-assessment
- **Mastery tracking** — tracks which terms you've reviewed (stored in browser)
- **Search** — instant filter across all terms
- **PDF version** — printable 16-page reference sheet

## Usage

Open `index.html` in any browser. No server, no dependencies, no setup — just a single HTML file.

Or use the [PDF version](ML_ENGINEER_GLOSSARY.pdf) for offline/print reference.

Anyone preparing for AI Engineer or ML Engineer interviews. Each term includes:
